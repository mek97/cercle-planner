# Code of Conduct

This is a small, friendly hobby project. Please treat it that way.

## In short

- **Be respectful.** Disagree about code, not people.
- **Assume good intent.** Most miscommunication is just miscommunication.
- **Keep it on-topic.** Issues and PRs are for the planner — not for
  off-topic debates, promotion, or harassment.
- **No hostility, harassment, or discrimination** of any kind, on any axis.

This applies to GitHub issues, pull requests, discussions, commit messages,
and any other space tied to this repo.

## Enforcement

If something here is being violated, email **mehul@earthsync.io** with a
link and a brief description. Reports are handled privately. Possible
responses range from a private nudge to blocking from the repo, depending
on severity and pattern.

The maintainer's decision is final for this project.
